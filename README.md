# Sign_Up_Form
A responsive split-screen sign-up page for a fictional coffee subscription brand, Redwood Coffee Co, made as part of The Odin Project curriculum.

This project focuses on layout structuring, form styling, validation states, and visual hierarchy using modern CSS techniques.

Project Overview:

- This is a front-end sign-up page featuring:
- Split-screen layout (image + content)
- Semi-transparent logo overlay
- Custom typography using @font-face
- Styled form inputs with focus states
- Live validation styling using :invalid
- Structured layout using Flexbox

Technologies Used:

- HTML5
- CSS3
- Flexbox

What I Practiced:
- Flexbox
- CSS Positioning
- HTML Forms and Form Validation
- CSS Pseudo Classes
